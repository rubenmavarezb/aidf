## Por que AIDF?

Assistentes de IA são poderosos, mas cegos ao contexto. Eles não conhecem sua arquitetura, convenções ou limites. O AIDF resolve isso.

<CardGrid>
  <Card title="Contexto Estruturado" icon="document">
    Defina a arquitetura, convenções e limites do seu projeto em um único arquivo `AGENTS.md` que qualquer assistente de IA pode entender.
  </Card>
  <Card title="Execução Automatizada" icon="rocket">
    Execute tarefas de forma autônoma com controle de escopo, validação, auto-commit e acompanhamento em tempo real pelo CLI.
  </Card>
  <Card title="Segurança Integrada" icon="approve-check-circle">
    O controle explícito de escopo previne alterações acidentais. Caminhos proibidos, caminhos permitidos e critérios de Definição de Pronto mantêm a IA focada.
  </Card>
  <Card title="Agnóstico de Ferramentas" icon="puzzle">
    Funciona com Claude Code, Cursor, GitHub Copilot ou qualquer LLM. O valor está no contexto estruturado, não na ferramenta.
  </Card>
</CardGrid>

## Início Rápido

```bash
# Instalar o CLI
npm install -g aidf

# Inicializar AIDF no seu projeto
aidf init

# Criar e executar uma tarefa
aidf task create "Add login form component"
aidf run .ai/tasks/add-login-form.md
```
